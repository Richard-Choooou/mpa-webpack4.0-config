# webpack4.0-config
基于webpack4.0的一套多页面脚手架配置

## 功能

集成以下功能
* vue
* typescript
* babel es6 转码 es5
* sass 转码 css
* 文件热更新
* 页面文件命令生成
* 抽离公共依赖
* etc

## 命令行

```cmd 
    npm run dev         //开发模式
```


```cmd 
    npm run build         //生产模式
```

```cmd 
    npm run new page1 页面一       
    //新建一个文件名为page1,title为‘页面一’的html页面,会生成相应的js入口和sass文件
```
